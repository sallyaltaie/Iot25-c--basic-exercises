# C++ Basic Exercises

This repository contains basic C++ exercises for learning and practicing programming concepts.

## Structure

- `main.cpp`, `main2.cpp`, `main3.cpp`: Example C++ programs
- `makefile`: Simple build instructions

## How to Build

To compile all programs, run:

```sh
make
```

To clean up compiled files, run:

```sh
make clean
```

## Requirements

- GCC or another C++ compiler
- Make
